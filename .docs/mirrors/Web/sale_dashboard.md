# Espejo: Web/sale_dashboard.html

## Objetivo
Presentar un análisis detallado de las ventas y el rendimiento financiero, facilitando la auditoría de transacciones y el análisis de métodos de pago.

## Lógica de Negocio
Funciona como el "Centro de Inteligencia Financiera". Agrupa las ventas por método de pago, vendedor y tienda, generando gráficos comparativos que permiten entender la salud del negocio de un vistazo. Incluye filtros avanzados para auditar periodos específicos con precisión forense.

## Interacciones
- **Gráficos**: Utiliza librerías como Chart.js para la visualización analítica.
- **Auditoría**: Se conecta con los registros de ventas de Blogger y AppSheet para una visión unificada.

## Valor para el Usuario (Criterio Publicitario)
- **Brújula Financiera**: Entiende exactamente de dónde viene cada peso y cómo está creciendo tu negocio.
- **Auditoría de un Vistazo**: Identifica tendencias y patrones en tus ventas sin necesidad de ser un experto en finanzas.
- **Decisiones Inteligentes**: Gráficos claros y precisos que te dan la confianza para invertir y escalar tu operación.
